# 📱 Campus Event – Android App

## 📌 Overview
Campus Event is an Android application designed to centralize university events into one accessible platform. It helps students easily discover and explore events without relying on scattered sources like emails or social media.

## 🎯 Features
- View upcoming campus events
- Access detailed event information
- Responsive UI (Portrait & Landscape)
- Smooth navigation between screens
- Dynamic event list using RecyclerView

## 🛠️ Technologies Used
- Kotlin
- Android Studio
- RecyclerView
- ConstraintLayout
- View-based UI

## Target Audience
- University students
- New students
- Transfer students
- Campus visitors

## 🚀 Future Improvements
- API integration for real-time events
- Search functionality
- Category-based filtering
